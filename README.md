# stock_market_analysis

A UI to present stock data of listed market

### data

1. profit_loss:

- 一般公司：
  - 營業收入 = 營業成本 + 營業毛利
  - 毛利率 ＝營業毛利/營業收入
- 銀行：
  - 營業收入 = 利息淨收益＋利息以外淨損益
  - 淨損 ＝營業收入 - 呆帳費用 - 營業費用
  - 沒有毛利
- 金控：
  - 營業收入＝淨收益 ＝利息淨收益＋利息以外淨收益
  - 沒有毛利
- 壽險:
  - 營業利益＝營業收入 - 營業成本 - 營業費用
  - 毛利率 ＝（營業收入-營業成本）/營業收入
- 其他：（可放棄）
  - 淨損 ＝ 收入 - 支出
